Sobre Mim

Olá! Meu nome é Carlos Henrique. Sou Desenvolvedor Full-Stack e de Jogos, atualmente aluno da Academia do Programador. Atuo na área há cerca de 4 a 5 anos, com grande dedicação aos estudos durante esse período.

Usando principalmente a Plataforma .NET com os frameworks MVC, Blazor e Web API, além de TypeScript usando Node.Js e Angular. Também uso Godot para desenvolvimento de jogos e com a Unreal Engine, explorando também a linguagem Verse.

Tenho experiência com versionamento de código utilizando Git, incluindo práticas como branching, merge e resolução de conflitos. Também utilizo o GitHub para colaboração em equipe, revisão de código, gerenciamento de issues e automações básicas com GitHub Actions, dentro de práticas fundamentais de DevOps.

Trabalho com bancos de dados relacionais, principalmente SQL Server e PostgreSQL, utilizando tanto ADO.NET quanto Entity Framework como ORM. Também utilizando tecnologias de containers tipo o Docker. Tenho domínio na criação e consumo de operações CRUD, implementação e controle de migrations para versionamento do banco de dados e manutenção da integridade dos dados.
